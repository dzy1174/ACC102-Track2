# ACC102 Track 2 - Profitability Analyzer
A Python data product for financial profitability analysis.

## Analytical Purpose
This project calculates and visualizes key financial ratios including gross margin, operating margin, net profit margin, and return on assets (ROA).

## Target User
Undergraduate students in Accounting, Economics, and Finance.

## Data Source
Sample financial data for reproducibility.
Date accessed: 26 April 2026

## How to Run
pip install pandas numpy matplotlib
python main.py

## Outputs
- Financial ratio dashboard
- Trend analysis charts
- CSV reports
- Excel reports
